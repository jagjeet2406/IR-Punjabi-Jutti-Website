## Deploying to Render

1. Push this repo to GitHub.
2. On Render, create a new **Web Service** and connect the repo.
3. Environment: **node**
4. Build command: `npm install && npm run build`
5. Start command: `npm start`
6. No environment variables were detected — add any your app needs.
7. Deploy. Render redeploys automatically on every push to the default branch.

This repo also includes a `render.yaml` — Render can pick this up automatically as a Blueprint instead of manual setup, via **New > Blueprint**.
